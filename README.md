# MinecraftLegacyDLC
Extracted contents of minercraft legacy leak dlc packages, after a certain leak of a certain game, we can have the source code but the DLC packages were stored in .pck / .arc files, not easy to view or use contents. This repo contains all of the extracted data of all those DLC in easy to use formats (.png, .txt, .json, etc.)

## Roadmap

- [x] Skin packs pck
- [x] Media arc files
- [x] x16 and x32 texture packs
- [x] Audio files
- [x] Gamerules headers
- [ ] Gamerules
- [x] Colors tables (note, the order of the channels may be wrong, I assumed ARGB stored as RGBA in file)
- [x] Localisation data
- [ ] SWF contents
